# Multi-Agent-Financial-Analysis-System

A multi-agent AI system built with **Phidata** and **Groq (LLaMA 3.3 70B)** that combines real-time web search and financial data analysis — accessible via both CLI and an interactive playground UI.

## 🧠 Agents

| Agent | Role | Tools |
|---|---|---|
| **Web Search Agent** | Searches the web for latest news and information | DuckDuckGo |
| **Finance Agent** | Retrieves stock data, fundamentals and analyst recommendations | YFinance |
| **Multi-Agent Coordinator** | Delegates tasks across both agents for combined responses | Both |
